# iTechDMV Website Design Style Guide

## Design Philosophy

### Visual Language
**Modern Government Technology Aesthetic**: Clean, professional, and trustworthy design that conveys expertise in government technology services. The design should evoke confidence in digital transformation while maintaining accessibility and user-friendliness.

### Color Palette
**Primary Colors**:
- **Deep Navy Blue (#1B365D)**: Primary brand color representing trust, stability, and government authority
- **Bright Teal (#00A6B8)**: Accent color representing innovation, technology, and forward-thinking
- **Clean White (#FFFFFF)**: Background and text contrast
- **Light Gray (#F8F9FA)**: Secondary backgrounds and subtle divisions

**Secondary Colors**:
- **Medium Gray (#6C757D)**: Secondary text and subtle elements
- **Success Green (#28A745)**: Success states and positive indicators
- **Warning Amber (#FFC107)**: Caution and attention states
- **Error Red (#DC3545)**: Error states and critical alerts

### Typography
**Primary Font**: "Inter" - Modern, clean sans-serif for excellent readability
**Secondary Font**: "Source Sans Pro" - Professional government-standard typography
**Display Font**: "Poppins" - Bold, modern headings with strong presence

**Hierarchy**:
- H1: 48px, Bold, Deep Navy
- H2: 36px, Semibold, Deep Navy  
- H3: 24px, Medium, Deep Navy
- Body: 16px, Regular, Dark Gray
- Caption: 14px, Regular, Medium Gray

## Visual Effects & Styling

### Hero Section Design
**Background**: Animated gradient mesh using teal and navy colors with subtle movement
**Text Treatment**: Large, bold typography with gradient color cycling effect
**Call-to-Action**: Prominent buttons with hover lift effects and subtle glow

### Interactive Elements
**Service Cards**: 
- Clean white cards with subtle shadows
- Teal accent borders on hover
- Smooth scale and shadow transitions
- Icon animations on card interaction

**Navigation**:
- Fixed header with backdrop blur effect
- Smooth underline animations for active states
- Mobile hamburger menu with slide transitions

**Forms**:
- Floating label inputs with teal focus states
- Progress indicators for multi-step forms
- Real-time validation with smooth feedback animations

### Animation Library Usage
**Anime.js**: 
- Smooth page transitions
- Staggered card animations
- Progress bar animations
- Button hover effects

**ECharts.js**:
- Service usage statistics
- Performance metrics visualization
- Interactive data dashboards

**Splide.js**:
- Service portfolio carousel
- Client testimonial slider
- Technology showcase gallery

### Background Effects
**Primary**: Subtle animated gradient mesh using brand colors
**Secondary**: Geometric pattern overlays with low opacity
**Accent**: Floating particle effects for technology theme

### Hover Effects
**Service Cards**: 3D tilt with shadow expansion and teal glow
**Buttons**: Color morphing with subtle scale increase
**Images**: Zoom with overlay text reveal
**Navigation**: Smooth underline animation with color transition

### Scroll Animations
**Reveal Timing**: Elements animate when 30% visible in viewport
**Stagger Delay**: 100ms between sequential elements
**Easing**: Smooth cubic-bezier transitions
**Duration**: 300ms for optimal performance

### Data Visualization Style
**Color Scheme**: Monochromatic teal variations with navy accents
**Chart Style**: Clean, minimal design with subtle grid lines
**Interactions**: Hover tooltips with smooth transitions
**Animation**: Smooth data loading with staggered reveals

## Mobile Design Considerations
- Touch-friendly button sizes (minimum 44px)
- Swipe gestures for service browsing
- Collapsible navigation with smooth animations
- Optimized form layouts for mobile input
- Reduced animation complexity for performance

## Accessibility Standards
- WCAG 2.1 AA compliance
- Minimum 4.5:1 contrast ratio for all text
- Keyboard navigation support
- Screen reader compatibility
- High contrast mode support
- Focus indicators for all interactive elements

## Brand Integration
- iTechDMV logo prominently displayed in header
- Consistent use of brand colors throughout
- Professional imagery reflecting government technology
- Clean, modern aesthetic matching government standards
- Trust indicators (certifications, partnerships) clearly visible
